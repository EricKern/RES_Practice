◆ DEFAULT_STACK_SIZE

#define DEFAULT_STACK_SIZE   32

Each task has its own stack to keep local variables and return addresses for function calls.

The default value supports very simple tasks like the ones presented in the example.

Use larger stacks for more complex stuff

Remember to NEVER use recursive functions in stack limited architectures (embedded systems) due to potenitally excessive stack usage
